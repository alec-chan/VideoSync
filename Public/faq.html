sync.theater


    sync.theater is a video sharing app, which allows you to watch videos in sync with your friends.

  • To get started, visiting the base address http://sync.theater/ which will generate a new room and redirect you to the room. By creating the room, you are automatically set as the owner, which means that you have full control over the video playback and can set the video.

  • To set the video, you need to paste a video URL into the input box next to the "Set URL" button. Currently, we only support youtube URLs and URLs that directly link to a video e.g. "http://XXXXX.com/XXXXX.mp4" etc...
